# StudyDesk 📚

A simple productivity app I built using Python to help students 
keep track of their tasks and notes in one place.

## What it does
- Add and manage your tasks
- Mark tasks as done
- Write and organize notes by subject
- Everything saves so you don't lose your data

## How to run it
1. Install Python
2. Download the files
3. Run: `python studydesk.py`
